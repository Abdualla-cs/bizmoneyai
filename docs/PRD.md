# Product Requirements (MVP)

## Goal
Help users track income/expenses, get rule-based insights, and test a category prediction endpoint.

## In Scope
- Auth with cookie-based JWT
- Categories and transactions CRUD
- Dashboard summary
- YAML-driven rule engine producing persisted AI insights
- Embedding-based category suggestion endpoint (MiniLM stub-capable)
- CSV import/export for transactions

## Out of Scope
- Multi-user organization/team features
- Extra database entities beyond required four tables
- Advanced ML training or model persistence
